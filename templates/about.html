ToDo Sense is simple and awesome web-app to organize your tasks with very easy to use interface. ToDo can help you to make list of your tasks and update or delete it when the task is accomplished.

Functions:

  • Add a ToDo by entering details and clicking on add todo button
  • Update a ToDo if you entered something wrong
  • Delete a ToDo when the task is accomplished

ToDo Sense is built using:

  • Python
  • Flask
  • SQLite
  • Flask SQLAlchemy
  • Jinja2 Template Engine
  • Bootstrap 5
